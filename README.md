# CDSL Operations Dashboard

Internal operations tracking dashboard for GEPL Capital Depository team.

## Stack
- React + Vite (frontend)
- Node.js + Express (backend)
- Google BigQuery (data)
- Google Cloud Run (hosting)

## Processes
- ST_BOD_FMS — File Download BOD FMS (13 steps)
- More FMS coming soon
